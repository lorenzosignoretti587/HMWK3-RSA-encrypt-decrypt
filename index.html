1. RSA Principle and Key Generation

RSA is an asymmetric encryption algorithm based on the difficulty of factoring large prime numbers. It requires two distinct keys: a Public Key for encryption and a Private Key for decryption.

For this demonstration, a simplified version is used. The key generation process is summarized below.

ParameterValueRole

Prime q11Another large prime number (simplified)
Modulus N (p · q)187Used in both Public and Private keys

Private Exponent d23Used for Decryption (The Private Key)
